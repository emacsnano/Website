TrustLens

Community-Powered Misinformation Defense Platform — developed for the HomeHack Hackathon

Overview

TrustLens is a Chrome browser extension that leverages community collaboration to empower citizens against fake news in real time. The extension automatically detects when users visit news websites, fetches their community reliability rating from a REST API, and displays visual indicators on the page. Users can submit their own ratings directly through the extension interface.

✨ Key Features

🌐 Real-time Rating

Detects news domains automatically and retrieves community reliability scores on a 1-10 scale.

👥 Crowdsourced Votes

Enables users to rate websites directly from the browser extension popup, feeding updates instantly to all users.

⚡ Seamless Extension

Lightweight Chrome background worker and content scripts overlaying visual reliability indicators directly on web pages.

🔌 Centralized API

Custom REST API backend designed to aggregate user votes, filter domains, and provide fast response endpoints.

🔌 REST API Endpoints

GET    /api/rating/:domain  — Fetch reliability rating for a specific domain
POST   /api/rating          — Submit a new score vote for a domain
GET    /api/domains/top     — Retrieve highest-rated trusted sites
GET    /api/domains/lowest  — Retrieve lowest-rated untrusted sites
GET    /health              — Verify backend system status
