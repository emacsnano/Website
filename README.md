# Owen Yap's Portfolio Website

Minimalist. 0 fluff. 100% me.

A typography-focused, lightweight, and interactive single-page portfolio showcasing my projects, hacker history, and tech credentials. Built entirely with vanilla web technologies.

[Live Site ↗](https://emacsnano.github.io/Website/)

---
## ✨ Premium UI/UX Features
This site is designed with modern aesthetic elements focused on micro-interactions and performance:
*   **Ambient Cursor Backlight**: A vanilla JS-tracked radial spotlight overlaying the dark layout.
*   **Glassmorphism Project Cards**: Compact, responsive grid cards that float and illuminate borders on hover.
*   **IntersectionObserver Scroll Reveals**: Sections smoothly slide and fade into view as visitors navigate.
*   **Zero-Framework Architecture**: Built with pure files—allowing 100-score performance metrics and instant loads.
