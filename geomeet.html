GeoMeet 🌍

Location-based Web3 networking app — developed for the NTU x Base Web 3 Hackathon

Overview

GeoMeet is a mobile-first, location-based social workspace application built for the NTU x Base Web 3 Hackathon. It helps active professionals coordinate offline meetups by showing nearby colleagues on a map, checking availability for lunch or after-office drinks, and settling payments securely using Coinbase OnchainKit and the Base Network.

Project Pitch Deck

✨ Key Features

📍 Real-time Location Map

Uses Leaflet and React-Leaflet to overlay interactive maps tracking nearby professionals with dynamic, real-time coordinate pins.

🤝 Availability Matching

Filter connections by active statuses ('lunch', 'after-office', 'unavailable') to find matching, nearby colleagues.

💳 Web3 & Onchain Payments

Integrates Coinbase OnchainKit, Wagmi, and Viem, allowing users to connect their Base wallets and send microtransactions using Base Pay.

🔔 Real-time Notifications

Features active messaging overlays and real-time toast alerts for newly incoming chat messages and meetup invites.

🗄️ Database Schema

-- Profiles table
create table profiles (
  id text primary key,
  name text,
  industry text,
  role text,
  food_preference text,
  availability text check (availability in ('lunch','after-office', 'unavailable')),
  lat double precision,
  lng double precision,
  contacts text[] default '{}'
);

-- Meetups table
create table meetups (
  id uuid primary key default gen_random_uuid(),
  participants text[] not null,
  type text check (type in ('lunch','after-office')),
  start_time timestamptz default now(),
  status text default 'ongoing',
  feedback jsonb default '{}'
);
